# Art Of Living Internship - Flutter UserLogin App
### What App Does ?
1. Login with e-mail.
2. Login with Google.
3. Show widget of various forms

### Challenge Faced:
1. Firebase connection.
2. Google sign-In.
3. Firebase database fetching & updating.
4. Keep user logged in even when app is closed.

### Widgets Used in app:
1. Alert Dialog
2. Rich Text 
3. Selectable Text 
4. Flexible 
5. Spacer 
6. Divider 
7. Ignore Pointer 
8. Colour Filtered 
9. ToolTip 
10. Aspect Ratio 
11. Sized Box
